Терминал:
1. В терминале прокручивать консоль shift + pgup, shift + pgdw.
2. Ctrl + Shift + c копирование в общий буфер.
3. Ctrl + Shift + v вставка.
4. Прокрутка недавно введённых команд Top or Buttom или Ctrl + p or Ctrl + n.
5. Перемещение в начало строки Ctrl + a.
6. Перемещение в конец строки Ctrl + e.
7. Удаление текущей строки Ctrl + u.
8. Удаление текста с текущей позиции до конца текста Ctrl + k.
9. Удаление слова, расположенного перед курсором Ctrl + w.
10. Поиск ранее введённых команд Ctrl + r.
11. Посылает сигнал завершения режима ввода Ctrl + c. Прекращает действие команды в терминале.
12. Закрыть текущий терминал Ctrl + d.
13. Команда history показывает команды которые были введённых ранее, до 1000. Использовать команду из списка  !5.
14. Команда clear помогает очистить от вывода предыдущих команд.
15. Команда touch созжает файл и предназначена для изменения последнего изменения файла или последнего времени доступа.
16. Команда mkdir позволяет создавать папки. Ключ -p позволит проигнорировать ошибки что такая папка уже есть.
17. Команда rmdir позволяет удалить папку без вложенности.
18. Команда rm позволяет удалить все. Для удаления файлов надо добавить ключ -r recursive, также можно добавить -f force что бы не показывало ошибок.
19. Команда cp путь путь позволяет скопировать файл. Ключ -v поможет получить информацию о ходе процесса. Ключ -r recursive позволяет копировать рекурсивно.
20. Команда mv путь путь позволяет переместить или переименовать файл или каталог. Если использовать mv -v *.txt ~/sample то перенесутся все файлы с расширением txt.
21. Команда cat ключи путь позволяет выводить на экран. Если не указывать ничего или указать ключ - то будет работа как с обычным редактором, что бы закончить надо нажать Ctrl+c or Ctrl+d. Если ввести cat file.txt > new_file.txt то создастся файл new_file.txt и записываются все данные с file.txt. Если ввести cat > new_file.txt то все данные которые будут вводится в терминале запишутся в файл.
22. Команда tac ключи путь позволяет вывести файл с конца.
23. Команда more позволяет просматривать файл целиком по размеру экрана, переключаться между страницами пробел, на одну строку enter, что-бы закончить нужно нажать Q или Ctrl+c.
24. Команда less ключи путь позволяет просматривать файл наподобие more, только более продвинуто.
25. Команда man команда позволяет открыть мануал по этой команде.
26. Команда find путь опции помогает найти файлы. Опции -name, -iname, -type, -size.
27. Команда which программа позволяет найти путь по которому расположена папка.
28. Команда grep ключи шаблон(строка) путь. Ключ -E позволяет указать что это расширенное регулярное выражение, -r рекурсивно, -n указывать номер строки, -T выравнивать табуляцией. Команда grep -rn get_id src позволяет найти строки где указана строчка get_id и вывести на экран. Проверить ключ -T выравнивание табуляцией. Служебные символы для шаблонов: "." - любой символ, "*" - множитель, преведущий символ может повторятся любое число раз, ".*" - любая последовательность символов, "^шаблон" - привязка шаблона к началу строки, "шаблон$" - привязка шаблона к концу строки, "[abc]" - указывает на любой символ указанный в квадратных скобках, (это под вопросом аналог "first_word\|second_word" заместь "|" - вертикальная черта указывается для перечисления искомых слов("первая|третья").
29. '|' - вертикальная черта позволяет передавать результат одной программы в другую. К примеру ls | grep .txt,  в начале выполниться команда лс в домашнем каталоге, а потом передаст информацию в команду grep.
30. Команда shutdown опции время сообщение позволяет выключить компьютер. Команда shutdown -r 18:05 [перезагружаемся] перезагрузит компьютер. Для перезагрузки лучше использовать shutdown -r now чем reboot так как shutdown позволяет закончить правильно скрипты.
